## 1. Critérios de seleção

- **Teto:** 25 artigos.
- **Distribuição:** 30% clássicos · 45% empíricos para discussão · 25% recentes (2023+).
- **Formato:** privilegiar artigos. Livros apenas para clássicos imprescindíveis (Hill Collins, Sueli Carneiro).
- **Fontes principais de reflexão interseccional:** Patricia Hill Collins e Sueli Carneiro (presença reforçada).
- **Função:** cada item deve ter uso claro no texto IMRaD (Introdução / Métodos / Resultados / Discussão / Conclusão). Sem item "decorativo".
- **Validação:** DOI confirmado para cada referência antes da entrega final.
